# Family Budget

Приложение для учета доходов и расходов семьи с аналитическим дашбордом.

Demo: https://pavellver.github.io/family-budget/

## Возможности

- Раздельный учет доходов и расходов с единым журналом операций.
- Добавление, редактирование и удаление записей.
- Фильтры по периоду: текущий месяц, прошлый, последние 30 дней, год, весь период, произвольный диапазон.
- Фильтры по категориям и поиск по описанию.
- Сортировка по дате и сумме, пагинация и выбор количества записей на странице.
- Аналитика за 12 месяцев: KPI, динамика доходов/расходов и прогноз накоплений на 6 месяцев.
- Графики в нескольких режимах (круг, столбцы, радар, динамика по времени).
- Импорт/экспорт JSON и Excel.
- При импорте можно выбрать: заменить текущие записи или дополнить их импортом.
- Кнопка загрузки демо-набора данных.
- Светлая и тёмная тема интерфейса.
- Все данные хранятся локально в `localStorage` (без бэкенда).

## Технологии

- React + TypeScript
- Vite
- Recharts
- SheetJS (`xlsx`)
- Tailwind CSS (через CDN в `index.html`)

## Требования

- Node.js 18+ (рекомендуется 20+)
- npm 9+

## Быстрый старт

```bash
npm install
npm run dev
```

После запуска открой адрес из консоли Vite (обычно `http://localhost:5173`).

## Локальная сборка

```bash
npm run build
npm run preview
```

Артефакты сборки будут в папке `dist/`.

## Тесты

```bash
npm test
```

Для запуска в watch-режиме:

```bash
npm run test:watch
```

## Где хранятся данные

- Все операции хранятся только в браузере (`localStorage`).
- Кнопки импорта/экспорта и демо-пресет находятся в меню приложения.
- Приложение работает полностью на клиенте, сервер не требуется.
