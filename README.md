# Family Budget

Приложение для учета доходов и расходов семьи с аналитическим дашбордом.

Demo: https://pavellver.github.io/family-budget/

## Возможности

- Раздельный учет доходов и расходов.
- Фильтрация по периоду и категориям.
- История операций с поиском, сортировкой и пагинацией.
- Аналитика за 12 месяцев: KPI, динамика, тренд накоплений.
- Импорт/экспорт в JSON и Excel.
- Локальное хранение данных в `localStorage` (без бэкенда).
- Кнопка загрузки демо-данных для быстрого теста интерфейса.

## Технологии

- React + TypeScript
- Vite
- Recharts
- SheetJS (`xlsx`)
- Tailwind CSS (через CDN в `index.html`)

## Требования

- Node.js 18+ (рекомендуется 20+)
- npm 9+

## Быстрый старт

```bash
npm install
npm run dev
```

После запуска открой адрес из консоли Vite (обычно `http://localhost:5173`).

## Локальная сборка

```bash
npm run build
npm run preview
```

Артефакты сборки будут в папке `dist/`.

## Тесты

```bash
npm test
```

Для запуска в watch-режиме:

```bash
npm run test:watch
```

## Где хранятся данные

- Все операции хранятся только в браузере (`localStorage`).
- Кнопки импорта/экспорта и демо-пресет находятся в меню приложения.
- Приложение работает полностью на клиенте, сервер не требуется.
